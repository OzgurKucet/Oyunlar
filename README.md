# Oyunlar


#KartOyunu:
Oyunun amacı 2 aynı kartı bulmak.
2 aynı kartı bulunca o kartlar açık kalacaklar ve tüm kartları açınca oyun biter.
Açmak istediğiniz kartın numarasını yazmanız gerek numaralar oyun başında belirtiliyor...
